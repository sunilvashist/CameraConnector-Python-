Open Camera and display it over window using python.

**Library Used**
 
1. cv2

2. numpy

**Steps**
1. **Python 3.6** or above version is must. 

2. To install all dependencies [ **requiremt.txt** ] type below command: 
    
    windows -->> pip install r- requirements.txt
    
3. Run **main.py**
